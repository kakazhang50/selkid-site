## 改什么 → 改哪里

| 目标 | 文件 |
|------|------|
| 口号、TPT、MailerLite、Formspree | `src/data/siteConfig.json` |
| 导航栏目 | `src/components/Header.astro` |
| 首页版块 | `src/pages/index.astro` |
| 新发 **Educator Note** | `src/content/educatorNotes/新建.md` |
| TPT 产品列表 | `src/data/catalog.json`（跑 `selkid-tools` 脚本后同步到 `public/data/`） |
| 全站样式 | `src/styles/global.css` |
| 换图 | `public/assets/` |

## 新发一篇 Educator Note

1. 在 `src/content/educatorNotes/` 新建 `my-topic.md`  
2. 顶部写：

```yaml
---
title: "标题"
description: "一句话摘要"
pubDate: 2026-06-30
featured: true
category: "classroom"
---
```

3. 正文用 Markdown  
4. `npm run dev` 预览 `/educator-notes`  
5. `npm run publish`

---

## MailerLite 配置

1. MailerLite 后台 → Forms → 嵌入代码，复制 `data-form="xxxx"`  
2. 填入 `src/data/siteConfig.json`：

```json
"mailerLiteAccountId": "账号数字",
"mailerLiteFormId": "表单短码"
```

3. `npm run publish`

未配置时，表单走 Formspree（`subscribeEndpoint`）或浏览器本地暂存。
